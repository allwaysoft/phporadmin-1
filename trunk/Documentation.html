Introduction

phpOrAdmin is based on phpMyAdmin 3.4.6. It is a simple tool that manage Oracle databases just like phpMyAdmin. But it does have only small part of the phpMyAdmin's features.

Currently phpOrAdmin can:

  • browse and drop databases, tables, views, columns and indexes
  • create, copy, drop, rename and alter databases, tables, columns and indexes
  • create1 and read dumps of tables
  • export1 data to various formats: CSV, XML, PDF, ISO/IEC 26300 - OpenDocument Text and Spreadsheet, Word, and LATEX formats

Installation

  1. Download the latest version of xampp that support Oracle and install it. Make sure that you can access the Oracle database through explorer.
  2. Download the latest release of phpOraAdmin. Extract the downloaded archive file to xampp installation directory, for example /opt/lampp/htdocs

Configuration

Edit config.inc.php file:

$cfg['Servers'][$i]['host'] string
The tnsname defined in tnsname.ora file.
$cfg['Servers'][$i]['extension'] string
oci : The Oracle oci extension.
